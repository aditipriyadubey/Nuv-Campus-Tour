// How It Works
🗺️

3D Weighted Graph

Campus is modeled as a graph where X=Block, Y=Location, Z=Floor. Each node is named Block_Floor_Location (e.g. A_6_LABS).

⚡

Dijkstra's Algorithm

Uses a min-heap priority queue to explore nodes in order of distance. Guarantees shortest path. O((V+E) log V) complexity.

🛗

Lift vs Stairs

L1/L2 travel at 5 sec/floor. S2 is fastest stair at 15 sec/floor. System compares all vertical movement options automatically.

🔁

Multi-Block Routes

Three inter-block routes: C→Admission→B→A (fast), C→NCC→MechLab→A (medium), C→Basement→A (alternate).
